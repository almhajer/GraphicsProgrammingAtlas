# Skill Router

استخدم هذا الملف فقط إذا لم تكن تعرف أي مهارة يجب قراءتها.

## كيف تختار المهارة
- إذا كانت المهمة عامة أو تشغيلية: `core-rules`
- إذا كانت المهمة تخص الشكل أو الترتيب أو اللغة أو CSS: `ui-style`
- إذا كانت المهمة تخص واجهة دردشة أو مساعد تفاعلي: `conversation`
- إذا كانت المهمة تخص البحث أو الكلمات المفتاحية أو الفهرسة: `search-indexing`
- إذا كانت المهمة تخص الربط بين المقالات والأمثلة والصفحات: `content-linking`
- إذا كانت المهمة تخص تقليل الحمل وتحسين الخفة: `performance-lightweight`
- إذا كانت المهمة تخص استكمال مكتبة C++ الحالية أو تحسين الشروحات والـ guides: `cpp-library`

## قاعدة مهمة
اقرأ مهارة واحدة فقط أولًا.
ولا تقرأ مهارة ثانية إلا إذا كانت المهمة فعلًا مشتركة بين الجانبين.
