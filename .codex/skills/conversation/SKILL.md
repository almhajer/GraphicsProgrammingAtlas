# conversation

## الغرض
إضافة دردشة تفاعلية تساعد المستخدمين في فهم المفاهيم البرمجية بالعربية، مع أمثلة وروابط داخلية، دون تغيير الهوية الحالية للموقع.

## القواعد الأساسية
- الحفاظ على الشكل الداكن والعربية كلغة أساسية
- استخدام JavaScript خفيف و lazy loading
- ربط المحادثة بالمحتوى الموجود مثل JSON والأمثلة
- عدم إضافة مكتبات ثقيلة؛ استخدم حلولًا محلية
- الحفاظ على الأداء على الأجهزة الضعيفة
- لا تجعل نظام المحادثة منفصلًا تمامًا عن محتوى الموقع

## عند إنتاج محتوى تعليمي عبر المحادثة
- اشرح بالمعنى الحقيقي لا الترجمة النصية
- اجعل الردود مناسبة للمبتدئ لكن بعمق حقيقي
- اربط الردود بأمثلة عملية وروابط داخلية مفيدة
