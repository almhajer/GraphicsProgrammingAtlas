# search-indexing

## الغرض
تحسين البحث الداخلي والوصول إلى الإجابات والصفحات والأمثلة المرتبطة بسرعة، مع تقليل استهلاك الملفات والسياق.

## القواعد الأساسية
- اقرأ فقط ملفات الفهرسة أو JSON المرتبطة بالمهمة
- لا تعِد قراءة كل ملفات البيانات
- استخدم exact-match أولًا عند وجود كلمة واضحة
- أضف fallback بسيطًا للبحث التقريبي
- حافظ على الأداء الخفيف
- تجنب الخوارزميات الثقيلة غير الضرورية
- لا تكسر الروابط الحالية أو بنية البيانات الحالية

## في المحتوى التعليمي المرتبط بالبحث
- اجعل الفهرسة والربط يخدمان المعنى الحقيقي للعناصر
- اربط النتائج بالعناصر القريبة منطقيًا لا بالاسم فقط
