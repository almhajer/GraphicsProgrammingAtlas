# ui-style

## الغرض
تحسين أو توسيع الواجهة دون كسر شكل المشروع الحالي، مع الحفاظ على العربية كلغة أساسية وعلى الطابع الداكن والتجربة الخفيفة.

## القواعد الأساسية
- حافظ على الشكل الداكن الحالي
- العربية هي اللغة الأساسية في النصوص والواجهات
- لا تكسر القائمة الرئيسية
- لا تغيّر التوزيع العام للصفحات إلا عند الضرورة
- اجعل العناصر واضحة ومحاذاتها منطقية
- قلل التكرار البصري والنصي
- لا تضف مكتبات واجهات ثقيلة إلا إذا كانت هناك ضرورة قصوى
- فضّل CSS بسيطًا ومنظمًا
- حافظ على الأداء وسرعة التحميل
