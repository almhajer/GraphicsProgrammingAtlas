# الترخيص

حتى الآن لا يوجد ترخيص مفتوح نهائي معلن لهذا المشروع.

جميع الحقوق محفوظة لصاحب المشروع، ولا يُعد وجود الشيفرة أو المحتوى داخل المستودع إذنًا تلقائيًا بإعادة الاستخدام أو إعادة التوزيع أو الاشتقاق أو النشر التجاري، ما لم يُعلن لاحقًا عن ترخيص صريح وواضح داخل المشروع.

إذا تقرر اعتماد ترخيص رسمي لاحقًا مثل `MIT` أو `Apache-2.0` أو غيرهما، فيجب استبدال هذا الملف بالنص القانوني الكامل للترخيص المعتمد.
