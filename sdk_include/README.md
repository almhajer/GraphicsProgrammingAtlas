# sdk_include

هذا المجلد مخصص فقط للاستخدام المحلي عند توليد صفحات ملفات ترويسات Vulkan من نسخة SDK مثبتة على جهاز المطور.

لا يجب تتبع الرابط الرمزي `sdk_include/vulkan` داخل git لأن GitHub Pages لا يملك نفس المسار المحلي، ووجوده يكسر بناء الموقع.

إذا احتجت إعادة توليد صفحات `pages/files` محليًا، أنشئ الرابط الرمزي أو اضبط مسار الترويسات محليًا على جهازك فقط من دون إضافته إلى المستودع.
